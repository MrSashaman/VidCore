# 🎬 VidCore

**VidCore** is a web-based video hosting platform built with ASP.NET Core Razor Pages.  
It provides user authentication, video and music uploading, social interactions (likes), comments, and content search.

---

## ✨ Features:

### 👤 Authentication System
- User registration and login
- Secure authentication flow
- Session management

### 🎥 Video System
- Upload videos
- Watch videos in browser
- Like videos
- Comment under videos
- Video search system

### 💬 Comments
- Comment system under videos
- User-based interaction
- Structured discussion under content

### 🎵 Music System
- Dedicated music panel
- Upload audio files
- Play music inside platform
- Like music tracks

### 🔎 Search
- Search videos by title/metadata
- Fast content discovery
- Integrated into UI

---

## 🛠 Tech Stack

- ASP.NET Core (Razor Pages)

- Entity Framework Core
- SQL Server / SQLite
- HTML5
- CSS3

## 🚀 Getting Started

### 1. Clone repository
```bash
git clone https://github.com/your-username/VidCore.git
```
### 2. Navigate to project
```bash
cd VidCore
```
### 4. Apply migrations (EF Core)
```bash
dotnet ef database update
```
### 5. Run project
```bash
dotnet run
```
